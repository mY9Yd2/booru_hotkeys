# Booru Hotkeys

## About

This browser extension adds several hotkeys to supported websites, such as keyboard-based pagination.

P.S. Don't forget to allow the extension to run in private windows if needed.

Currently implemented hotkeys (for all supported websites):

- Pagination: Use the Left/Right arrow keys to navigate between pages. Lowercase 'a' and 'd' are also supported.

Currently supported websites (some of them are NSFW):

- rule34.xxx
- gelbooru.com
- safebooru.org
- xbooru.com
- fanservice.fan
- zerochan.net

## Install

- Firefox Addons: [booru-hotkeys-next](https://addons.mozilla.org/en-US/firefox/addon/booru-hotkeys-next/)
- Chrome Web Store: not published yet

## Dev/Local install

Firefox (temporary):

1) Visit the about:addons page
2) Gear icon -> Debug Add-ons
3) Load Temporary Add-on... and select the 'manifest.json' from the booru_hotkeys folder

In some cases, you may need to enable the optional permissions under the 'Permissions and data' tab.

Google Chrome or chrome like browsers (seems permament):

1) Visit the chrome:extensions page
2) Enable 'Developer mode' (top right corner)
3) 'Load unpacked' button and select the booru_hotkeys folder
